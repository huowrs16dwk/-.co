connecting to github securely

First make sure you have tor running. Either start the daemon process or run it manually with tor

	sudo systemctl start tor

$ cat ~/.ssh/config
Host github.com
    User git
    IdentitiesOnly yes
    ProxyCommand nc -X 5 -x 127.0.0.1:9050 %h %p
